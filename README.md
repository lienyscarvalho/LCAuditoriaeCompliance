# LC Consultoria e Auditorias

Site institucional completo — **LC Consultoria e Auditorias** · Rio de Janeiro

## 🌐 Deploy
Faça upload do `index.html` no GitHub e conecte à Vercel para deploy automático.

## 📄 Páginas incluídas
| Página | Descrição |
|---|---|
| Home | Hero, Dashboard GRC, Cases de Sucesso, Compliance, Riscos, Auditoria, Normas, Contato |
| Sobre | Equipe, história e valores da LC Consultoria |
| Blog | 12 artigos com imagens reais sobre Compliance, Riscos, Auditoria e LGPD |
| Política de Privacidade | Documento completo conforme LGPD |
| Termos de Uso | Condições gerais detalhadas |

| ISO 37001 | Sistema de Gestão Antissuborno |

## 🤖 IA Consultora
O chat de IA usa a API da Anthropic. Para ativá-lo em produção, insira sua chave no campo da requisição fetch no script do chat (busque por `api.anthropic.com`).
